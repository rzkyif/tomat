# Pre-send checklist

Before sending a web-research answer, confirm:

- [ ] Every time-sensitive claim came from a page read this turn, not from prior knowledge.
- [ ] At least two independent sources agree on each key fact.
- [ ] The answer leads with the conclusion, in your own words.
- [ ] Sources are listed as links at the end.
- [ ] Any uncertainty or source disagreement is stated plainly.
