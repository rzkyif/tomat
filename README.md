# Tomat

**Open source local-first modular AI assistant.**

Tomat takes care of all the hassle in running a complete state-of-the-art local
AI assistant on your PC, with speech-to-text, LLM inference, text-to-speech, and
a bunch of other quality of life features.

## Core Principles

- **Local-first: privacy, security, and minimal resource usage.** Everything
  stays on your machine by default, with options for those who want to use
  external providers. The app itself follows standard security best practices
  and is optimized to work in most PCs.
- **All-in-one, batteries included.** Designed to just work after install, while
  staying deeply customizable for more specific use cases and preferences with a
  robust configuration system.
- **Modular and extensible.** The system is separated into modules that can be
  completely disabled when not used. The toolkit system provides an easy and
  secure way to quickly provide additional tools to the assistant.

## Getting Started

The latest client can be downloaded from the official website at
